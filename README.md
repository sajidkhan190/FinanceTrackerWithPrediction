# FinanceTrackerWithPrediction
VU Final Project for CS-619

A python-based web application that helps users manage income and expenses, visualize financial trends, and predict future expenses using Machine Learning.

# Tech Stack

    * Backend: Flask (Python)
    * Database: SQLite
    * Machine Learning: Scikit-learn
    * Visulaization: Matplotlib / Plotly
    * Frontend: HTML, CSS, JavaScript, Bootstrap
    * Version Control: Git & GitHub
